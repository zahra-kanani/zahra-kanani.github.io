# zahra-kanani.github.io (al-folio minimal)

Minimal, GitHub Pages–friendly configuration using the al-folio theme.

## How to deploy
1. Create a public repo named `zahra-kanani.github.io`.
2. Upload these files (or extract the ZIP into the repo).
3. In **Settings → Pages**: Source = Deploy from a branch → `main` → root (`/`).
4. Wait 1–2 minutes, then open https://zahra-kanani.github.io

## Edit content
- Navigation: `_data/navigation.yml`
- Pages: `_pages/*.md`
- CV file: `assets/Zahra_Kanani_CV.pdf`
